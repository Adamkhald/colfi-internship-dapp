# 🏦 Colfi Internship Project – Blockchain Investing & Collateral Management App

This project is a **blockchain-powered web application** developed as part of my internship at **Colfi**. It enables users to securely manage investments and collateral using a **permissioned blockchain network** built with **MiniFabric (Hyperledger Fabric)**. The frontend is built with **Next.js**, **Node.js**, and modern web development tools for a seamless user experience.

---

## 🚀 Features

- 🔐 **Secure Investment Workflows**  
  Users can initiate and manage investments via blockchain-backed smart contracts.

- 💰 **Collateral Management**  
  Deposit, withdraw, and track collateralized assets on a permissioned ledger.

- 🧱 **Enterprise Blockchain Integration**  
  Powered by **MiniFabric** (Hyperledger Fabric), ensuring trust and transparency.

- 🖥️ **Modern Web Frontend**  
  Built using **Next.js**, **Node.js**, **TailwindCSS**, and other frameworks.

- 🧩 **Modular Smart Contracts**  
  Business logic is encapsulated in smart contracts for easy updates and auditing.
